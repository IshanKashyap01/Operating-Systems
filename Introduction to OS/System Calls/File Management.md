# File Management

This category of system calls deals with operations and security of a file

## Operations

- Open
- Read
- Write
- Close
- Create

## Security

- Ownership of a file
- Permissions regarding read, write and execution of a file

## System Calls in File Management

### For Operations

- Open()

- Read()

- Write()

- Close()

- Create()

### For Security

- chown()

- chmod()
